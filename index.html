🔍 WHAT IT DOES (SEARCH ONLY)

  • Searches %APPDATA%, %LOCALAPPDATA%, %TEMP%, %ProgramFiles%, %ProgramData%, %USERPROFILE%\Documents, Downloads, Desktop for files/folders containing "superiority" → reports them (no deletion).
  • Checks registry keys: HKCU\Software\Superiority, HKLM\Software\Superiority, HKCU\Software\Classes\superiority and autorun value.
  • Checks USN Journal on C: D: E: F: G: (if exist).
  • Checks Windows Event Logs: NTFS/Operational, System, Security, Application.
💡 Note: Administrator rights are NOT required for basic file/registry search, but ARE required for USN Journal and Event Log checks.
🔒 The script will warn you if run without admin — you can still continue, but some checks will be skipped.
📄 Output can be copied/saved as evidence for moderation.

❓ TROUBLESHOOTING

  • "Python not recognized" → Install Python from python.org (add to PATH).
  • Some checks show "error" → Run as Administrator for full access to USN/Event Logs.
  • No traces found → The user may not have "superiority" cheat installed, or it uses different naming.
  • False positives? → The script only matches files/folders containing "superiority" (case-insensitive). Verify manually if needed.
