INSTALLATION & RUN

Requirements: Windows 10/11, Python 3.6+ (for .py) or no requirements (for .exe)

FOR MODERATORS ONLY
This tool does NOT delete anything. It only scans and reports traces of "superiority" cheat.
Use results as evidence. No system changes are made.

METHOD 1 — via website (easiest)

  1. Click the DOWNLOAD button above and choose your preferred version.
  2. .exe: right-click → Run as administrator. No Python needed.
    .py: right-click → Open with Python, or run python "fiNdIT!.py" in CMD.
  3. Administrator rights are recommended for full search (USN/Event Logs).
  4. Press ENTER and wait ~2-3 minutes.
  5. At the end, confirm saving the report to Desktop as a .txt file for evidence.

METHOD 2 — via PowerShell (iwr)

# Run in PowerShell (admin recommended): $url = "https://github.com/atlasru/findit-superiority/download/fiNdIT!.py" $out = "$env:TEMP\fiNdIT!.py" Invoke-WebRequest -Uri $url -OutFile $out python $out

METHOD 3 — Git clone

git clone https://github.com/atlasru/findit-superiority.git cd findit-superiority/download python "FindIT!.py"

WHAT IT CHECKS

  • File system: %APPDATA%, %LOCALAPPDATA%, %TEMP%, %ProgramFiles%, %ProgramData%, Documents, Downloads, Desktop
  • Registry: HKCU\Software\Superiority, HKLM\Software\Superiority, autorun entries
  • Execution traces: Prefetch, UserAssist, AppCompatFlags, Recent shortcuts
  • USN Journal status on C: D: E: F: G:
  • Event Logs: NTFS/Operational, System, Security, Application
  • cLeanIT! cleaner evidence: script files, cleaner logs, Defender logs
  • System info: PC name, HWID, region, OS version (for ban report)
Administrator rights are NOT required for basic file/registry search, but ARE required for USN Journal and Event Log checks.
The script will warn you if run without admin — you can still continue.
At scan end, you will be prompted to save the full report as .txt for moderation evidence.

TROUBLESHOOTING

  • "Python not recognized" — Install Python from python.org (add to PATH), or use the .exe release.
  • Some checks show error — Run as Administrator for full access to USN/Event Logs.
  • No traces found — The user may not have Superiority installed, or it was cleaned. Check cLeanIT! evidence section.
  • Antivirus flags the .exe — False positive due to PyInstaller packaging and system scan nature. Source code is available for verification.

FOR MODERATORS

This tool is designed for evidence gathering only. It does not:

  • Modify the registry
  • Clear logs or journals
  • Change any system state

Run it on the target machine, save the report, and attach as proof for the ban decision.
